\section{\makebox[\widthof{\faUsers}][c]{\color{CVBlue}\faUsers}\ 项目经历}
\textbf{具身 VLM--VLA 能力关联评测基准}
\hfill 2026.05 -- 至今 \titlebreak
项目描述：基于机器人操作数据构建具身 VQA 基准，评测 VLM 的任务理解、空间感知与动作相关推理能力，并研究其与采用对应视觉语言骨干的 VLA 操作表现之间的相关性。
\begin{itemize}[nosep, topsep=\intraProjectListTopSep]
    \item \textbf{任务构建：}面向单视角、多视角和历史序列三类输入构建 VQA 任务，涵盖高层任务规划、低层轨迹选择、抓取位姿决策以及空间与环境感知等 10 类任务。
    \item \textbf{数据筛选：}从 DROID、AgiBot World、RoboMIND 和 RoboTwin 中挖掘机器人操作数据，基于亮度、对比度和模糊程度进行视频抽帧初筛；使用 Qwen3.5-35B-A3B 判断样本与任务的匹配性，并对筛选结果进行人工复核。
    \item \textbf{QA 生成：}优先利用原始数据集状态与轨迹信息生成答案和干扰项；空间感知任务采用 VLM 进行目标定位，并结合 Grounded SAM 生成区域标注；抓取任务使用 GraspGen 生成候选抓取位姿。
    \item \textbf{评测分析：}通过主流闭源 VLM 和人工测试验证数据可靠性；设计开源 VLM 与对应 VLA 的配对测试，按任务类别和输入类型分析 VLM 能力与 VLA 操作表现之间的相关性。
\end{itemize}

\textbf{基于人类视频的跨具身动作提取与 VLA 微调}
\hfill 2026.01 -- 至今 \titlebreak
项目描述：从人类操作视频中恢复手部三维运动并映射为机器人末端动作，利用机器人动作先验对映射轨迹进行分布校正，并将处理后的动作数据用于 VLA 微调。
\begin{itemize}[nosep, topsep=\intraProjectListTopSep]
    \item \textbf{人手动作提取：}使用 HaMeR 提取手部关键点，结合 Depth Anything V2 深度图计算手部三维位姿，并通过 Grounded SAM 分割任务相关物体；在样条轨迹优化中加入机器人执行速度惩罚项，生成机器人末端动作。
    \item \textbf{轨迹分布校正：}将人手轨迹映射为机器人末端动作表示，使用基于机器人示范动作预训练的条件扩散模型对跨具身映射轨迹进行分布校正，并设置低通滤波和直接映射作为对比基线。
    \item \textbf{VLA 微调：}使用人类视频中提取的动作数据分别对 OpenVLA 和 $\pi_0$ 进行 LoRA 微调，并在 LIBERO 中开展闭环操作评测。
\end{itemize}

\textbf{基于末端力控的机械臂柔顺轨迹跟踪系统} \hfill 2024.05 -- 2025.05 \titlebreak
项目描述：在六自由度机械臂平台上实现柔顺控制，并能够对未知曲面进行跟踪；在此基础上，基于触觉信息使用 LSTM 架构实现夹持柔性工具的轨迹跟踪。
\begin{itemize}[nosep, topsep=\intraProjectListTopSep]
    \item \textbf{柔顺控制：}使用 ROS 部署传感器与机械臂平台通信，基于厂商 SDK 与控制接口，完善 ROS 框架下的机械臂位置与速度控制，编写导纳控制算法并在实机上完成验证。
    \item \textbf{数据与模型：}设计数据收集实验方法，在力位控制条件下使用触觉传感器采集夹爪夹持柔性工具的触觉信息，作为模型的时序特征输入；选取 ACTPP 作为时序模型框架构建触觉信息与环境接触特征之间的关系，并利用预测的接触力信息实现跟踪柔顺控制。
\end{itemize}
